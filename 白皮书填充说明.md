# 从PDF填充白皮书网页内容

## 快速开始

### 方法1：使用批处理文件（推荐）

双击运行：
```
填充白皮书.bat
```

### 方法2：手动运行Python脚本

```bash
python fill_whitepaper_from_pdf.py
```

## 功能说明

脚本会自动：
1. ✅ 安装必要的Python库（pdfplumber）
2. ✅ 从 `PowerVerseDecentralizedComputingWhitepaper-zh.pdf` 提取文本内容
3. ✅ 解析PDF内容，识别各个章节：
   - 引言
   - 项目概述
   - 技术架构
   - 代币经济
   - 路线图
   - 团队介绍
4. ✅ 将提取的内容格式化为HTML并填充到 `pages/whitepaper.html`

## 输出文件

- `whitepaper_content.txt` - PDF的原始文本内容（用于参考）
- `pages/whitepaper.html` - 更新后的白皮书网页

## 注意事项

1. **自动解析可能不完美**：PDF格式复杂，自动解析可能无法100%准确识别章节边界
2. **手动调整**：运行脚本后，请检查 `pages/whitepaper.html`，根据需要手动调整内容
3. **格式优化**：脚本会自动将文本格式化为HTML段落和列表，但可能需要进一步优化

## 如果遇到问题

1. **无法安装pdfplumber**：
   ```bash
   pip install pdfplumber
   ```

2. **PDF文件不存在**：
   - 确保 `PowerVerseDecentralizedComputingWhitepaper-zh.pdf` 文件在项目根目录

3. **解析不准确**：
   - 查看 `whitepaper_content.txt` 了解PDF原始内容
   - 手动编辑 `pages/whitepaper.html` 进行精确填充

## 手动填充指南

如果自动填充效果不理想，可以：

1. 打开 `whitepaper_content.txt` 查看PDF内容
2. 打开 `pages/whitepaper.html` 
3. 找到对应的章节（如 `<h2>引言</h2>`）
4. 将PDF内容复制粘贴到对应的 `<p>` 或 `<ul>` 标签中
5. 保持HTML格式正确
