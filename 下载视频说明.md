# 视频下载说明

## 云游戏视频

视频URL: `https://image-hibaocloud.oss-cn-hangzhou.aliyuncs.com/cli-web/images/video.mp4`

保存路径: `images/scenarios/videos/cloudgaming.mp4`

## 下载方法

### 方法1: 使用浏览器下载
1. 打开浏览器，访问视频URL
2. 右键点击视频，选择"另存为"
3. 保存到 `images/scenarios/videos/cloudgaming.mp4`

### 方法2: 使用Python脚本
运行以下Python脚本：

```python
import urllib.request
import os

# 创建目录
os.makedirs("images/scenarios/videos", exist_ok=True)

# 下载视频
url = "https://image-hibaocloud.oss-cn-hangzhou.aliyuncs.com/cli-web/images/video.mp4"
save_path = "images/scenarios/videos/cloudgaming.mp4"

print("正在下载云游戏视频...")
urllib.request.urlretrieve(url, save_path)
print(f"✅ 下载完成: {save_path}")
```

### 方法3: 使用curl (如果已安装)
```bash
curl -o images/scenarios/videos/cloudgaming.mp4 https://image-hibaocloud.oss-cn-hangzhou.aliyuncs.com/cli-web/images/video.mp4
```

## 注意事项

- 如果视频下载失败，页面会自动使用在线URL作为备用
- 视频文件较大，下载可能需要一些时间
- 确保 `images/scenarios/videos/` 目录存在
