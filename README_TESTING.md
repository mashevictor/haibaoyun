# 测试部署指南

## 快速开始

### 方式1：一键部署测试（最简单）

直接双击运行：**`部署并测试.bat`**

这会自动：
- ✅ 启动HTTP服务器
- ✅ 运行所有测试
- ✅ 生成测试报告

### 方式2：分步操作

#### 第一步：启动服务器

双击运行：**`启动测试服务器.bat`**

或者命令行：
```bash
python start_server_for_test.py
```

服务器地址：http://localhost:8000

#### 第二步：运行测试

在**新窗口**双击运行：**`运行测试.bat`**

或者命令行：
```bash
npm test
```

#### 第三步：查看报告

```bash
npm run test:report
```

或打开：`playwright-report/index.html`

## 测试内容

✅ 多语言翻译检查（英文环境下是否有中文）  
✅ 空链接检查  
✅ 语言切换器功能测试  
✅ 用户操作模拟（点击、滚动）
