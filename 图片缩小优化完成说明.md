## 🚀 如何启动服务

### 方法1：使用Python脚本（推荐）
```bash
python 自动压缩并启动服务.py
```

### 方法2：使用批处理文件（Windows）
双击 `批量压缩并启动服务.bat`

### 方法3：手动启动
```bash
# 1. 压缩图片
python compress_media.py

# 2. 启动服务器
python -m http.server 8000
```
