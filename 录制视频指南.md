# 📹 PowerVerse Chain 网站录制视频指南

## 🚀 快速开始

### 方法1：使用自动脚本（推荐）
1. **双击 `自动打开并录制.bat` 文件**
2. 脚本会自动：
   - 启动Web服务器
   - 打开浏览器访问网站
   - 显示录制提示

### 方法2：手动启动
1. 双击 `打开网站.bat` 启动服务器
2. 在浏览器访问 `http://localhost:8000/index.html`
3. 使用录制工具开始录制

---

## 🎬 Windows 自带录制工具

### Xbox Game Bar（Windows 10/11 自带）

**开始录制：**
- 按 `Win + G` 打开 Xbox Game Bar
- 点击红色录制按钮
- 或按 `Win + Alt + R` 直接开始录制

**停止录制：**
- 按 `Win + Alt + R` 停止录制
- 或点击 Xbox Game Bar 中的停止按钮

**录制设置：**
- 按 `Win + G` 打开设置
- 可以调整录制质量、音频等

**视频保存位置：**
- 默认保存在：`C:\Users\你的用户名\Videos\Captures\`

## 🎥 专业录制工具推荐

### 1. OBS Studio（免费，推荐）
- **下载地址：** https://obsproject.com/
- **优点：** 免费、开源、功能强大、支持直播
- **适合：** 专业录制、直播、多场景切换

**使用步骤：**
1. 下载并安装 OBS Studio
2. 打开 OBS，添加"显示器采集"源
3. 点击"开始录制"
4. 录制完成后点击"停止录制"

### 2. ShareX（免费，开源）
- **下载地址：** https://getsharex.com/
- **优点：** 免费、轻量、支持截图和录屏
- **适合：** 快速录制、截图、GIF制作

### 3. Bandicam（付费）
- **下载地址：** https://www.bandicam.com/
- **优点：** 轻量、高质量、易用
- **适合：** 游戏录制、屏幕录制

   - 展示各个产品卡片
   - 说明产品功能

5. **应用场景页面** (30秒)
   - 访问 scenarios.html
   - 展示各个应用场景

6. **代币页面** (20秒)
   - 展示代币信息
   - 展示RPC和钱包连接信息

7. **硬件工厂页面** (20秒)
   - 展示产品中心
   - 展示公司资质

8. **研发实力页面** (20秒)
   - 展示研发能力
   - 展示团队信息

9. **响应式设计演示** (20秒)
   - 调整浏览器窗口大小
   - 展示移动端适配

**总时长建议：** 2-3分钟

## ⚙️ 录制设置建议

### 分辨率：
- **推荐：** 1920x1080 (1080p)
- **最低：** 1280x720 (720p)

### 帧率：
- **推荐：** 30 FPS
- **流畅：** 60 FPS（如果电脑性能允许）

### 音频：
- 如果需要解说，确保麦克风已连接
- 可以关闭系统声音，只录制麦克风

### 文件格式：
- **推荐：** MP4（兼容性好）
- **高质量：** MOV（Mac）或 AVI

## 🎯 录制技巧

## ❓ 常见问题

**Q: 录制时卡顿怎么办？**
A: 降低录制分辨率或帧率，关闭其他程序

**Q: 录制文件太大怎么办？**
A: 使用视频压缩工具，或降低录制质量

**Q: 如何只录制浏览器窗口？**
A: OBS Studio 可以选择"窗口采集"，只录制浏览器

**Q: 录制没有声音？**
A: 检查音频设置，确保选择了正确的音频源
