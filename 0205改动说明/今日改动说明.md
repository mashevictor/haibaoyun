# 2025年2月5日 今日改动说明

本文档记录当日对嗨豹云官网代码的改动内容，便于回溯与部署核对。

## 一、产品生态 3D 效果调整

### 1. 开发者中心背景恢复
- **文件**：`styles.css`
- **说明**：开发者中心（`.developer`）背景恢复为之前的深色渐变样式。
- **具体**：
  - 背景：`linear-gradient(180deg, #0a0e27 0%, #0d1229 40%, #080c1a 100%)`
  - 恢复顶部 3px 青色高光线（`::before`）
  - 恢复底部左侧浅蓝径向光晕（`::after`）

### 2. 产品生态 3D 粒子底图（Three.js）
- **文件**：`index.html`、`styles.css`
- **说明**：产品生态区块底图改为使用 Three.js 渲染的粒子效果。
- **具体**：
  - 在「产品生态」区块内新增画布：`<canvas id="products-three" class="products-three-canvas">`
  - 移除原 CSS 实现的 `.products::after` 粒子层
  - 新增样式：`.products-three-canvas`（绝对定位、铺满、不阻挡点击）

### 3. 产品生态：波浪粒子 3D 效果
- **文件**：`index.html`
- **说明**：将产品生态的 3D 效果改为「波浪粒子」：粒子网格随正弦波起伏并缓慢旋转。
- **具体**：
  - 粒子网格约 100×65，Y 轴位置按 `sin(x·k1 + t)` 与 `sin(z·k2 + t)` 叠加形成波浪
  - 整组粒子绕 Y 轴缓慢旋转
  - 使用 `IntersectionObserver` 仅在区块进入视口时运行动画，离开视口停止，以节省性能

## 二、代币页多语言适配（token.name）

### 问题
- 代币页（如 `pages/token.html`）中「PEC 代币」名称（`data-i18n="token.name"`）未在多语言文件中定义，导致不显示对应语言文案。

### 修改
- **文件**：`js/languages.js`、`pages/token.html`
- **说明**：为五种语言补充 `token.name`，并统一 token 页默认文案。
- **具体**：
  - **zh-CN**：`token.name` = `PEC 代币`
  - **zh-TW**：`token.name` = `PEC 代幣`
  - **en**：`token.name` = `PEC Token`
  - **ja**：`token.name` = `PEC トークン`
  - **ko**：`token.name` = `PEC 토큰`
  - `pages/token.html` 中对应元素的默认文案由 `PEC Token` 改为 `PEC 代币`
