### 3. 批处理文件
- **文件**: `压缩图片.bat`
- **说明**: 双击运行即可压缩所有图片

## 如何运行图片压缩

### 方法1：使用批处理文件（推荐）
1. 双击 `压缩图片.bat`
2. 等待压缩完成
3. 压缩后的图片保存在 `images/compressed/` 目录

### 方法2：使用Python命令
```bash
python compress_media.py
```

## 压缩参数

- **图片最大尺寸**: 1200x600 像素
- **图片质量**: 80%
- **格式转换**: PNG自动转换为JPG（减小文件大小）

## 注意事项

1. **已压缩的图片**: 脚本会自动跳过 `compressed` 目录中的文件，避免重复压缩
2. **原始图片**: 原始图片不会被删除，压缩后的图片保存在 `images/compressed/` 目录
3. **HTML引用**: 所有HTML文件已更新为使用压缩后的图片路径（`images/compressed/...`）
